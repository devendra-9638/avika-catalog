# Avika Catalog

A simple, creative, and responsive **Product Catalog Web App** built with **Flask (Python)**.  
Users can upload products with multiple images, view a beautiful catalog, and see detailed product pages with image carousel.

## ✨ Features
- **Add Products**: Upload name, price, and multiple images (PNG, JPG, JPEG, GIF, WebP supported)  
- **Catalog View**: Modern grid of product cards with hover effects and zoom  
- **Product Details**: Image carousel + clean price display  
- **Responsive Design**: Works great on mobile, tablet, and desktop (Bootstrap 5)  
- **Flash Messages**: Success/error notifications for better UX  
- **JSON-based Storage**: Simple `products.json` for data (easy to backup & edit)

## 🛠️ Tech Stack
- **Backend**: Flask (Python)  
- **Frontend**: Bootstrap 5 + Custom CSS (gradients, shadows, animations)  
- **Storage**: JSON file + Static uploads folder  
- **Deployment Ready**: Render.com, PythonAnywhere, Railway, etc.
